## Phase 1: Bot vs Agent 术语统一

**原则：**
- ANI（IM 层）只用 **Bot** — 通信实体，有 token，收发消息
- Agent 是 AI 层概念（OpenClaw），不出现在 ANI 的 UI/API/文档中
- 后端 entity_type 保持 `bot`/`service`/`user` 不变

- [x] 1.1.3 Onboarding guide: "Agent 接入" → "Bot 接入"
